<h2 id="descricao">:pencil2: Descrição</h2>
Site onde um desenvolvedor pode se cadastrar e inserir as tecnologias dominadas por ele.

<h2 id="funcionalidades">:hammer: Funcionalidades</h2>

- Cadastro e login.
- Validação de dados inseridos no formulário de cadastro.
- Inserção e remoção das tecnologias dominadas pelo programador.
- Interação com API externa.
- Resposividade.
